# Kiosk SDK

Kiosk SDK is a tool to easily interact with Sui/Mysten Kiosk. You can use it to quickly query kiosk
related data, easily build transactions to interact, and build on top of it to support custom rules.

## Installation

```sh npm2yarn
npm i @mysten/kiosk
```

## About

Kiosk SDK is split into two logical clients, the `KioskClient` and the `TransferPolicyManager`.

- [Read more about `KioskClient`](/kiosk/kiosk-client/introduction)
- [Read more about `TransferPolicyManager`](/kiosk/transfer-policy-manager/introduction)
